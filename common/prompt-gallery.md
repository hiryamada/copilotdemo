# 「Copilot プロンプトギャラリー」を利用する


## PowerPoint

- 画面上部のメニューのCopilotボタンをクリック
- プロンプトを送信し、送信したプロンプトにマウスカーソルを合わせる。「プロンプトの保存」をクリック
- 画面右側のプロンプト入力部の左の「＋」をクリック
- 「プロンプトを表示する」をクリック
- Microsoftが提供するプロンプトと、ユーザーが作成したプロンプトをここで選択（再利用）できる。

## Microsoft 365 Copilot Chat

- プロンプトを送信し、送信したプロンプトにマウスカーソルを合わせる。「プロンプトの保存」をクリック
- 「プロンプトを表示する」をクリック
- Microsoftが提供するプロンプトと、ユーザーが作成したプロンプトをここで選択（再利用）できる。
