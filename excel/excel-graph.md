
# Excel: グラフの作成

- Excelに移動
- 「空白のブックを作成」をクリック
- 以下の表データをコピーしてシートに貼り付け
- Copilotボタンをクリック、「折れ線グラフを作成」
- 「＋新しいシートに追加」をクリック

表データ(タブ区切り):
```
年度	売上目標
2026	1000
2027	1200
2028	2000
2029	4500
```
